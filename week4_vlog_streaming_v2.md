## [8-22분] Target Expressions — 오프닝/클로징 멘트

### 🟡 문화 코칭 (Why moment)

**SPLIT** (영어):
```
"Korean content creators have a very specific vibe.

 The typical K-YouTuber/TikToker opening is NOT:
 '안녕하세요 제 이름은 OO입니다.' (too formal, sounds like a job interview)

 Instead, they say things like:
 '안녕 예쁜이들~' (Hey, pretty ones~)
 '다들 잘 지냈어?' (Have you all been well?)

 It's INTIMATE. Like they're FaceTiming their best friend.
 That's the energy we want for your channel.

 Also: Korean fans LOVE it when a foreigner uses
 Korean internet slang. It shows you're 'one of us.'
 We're going to build your signature opener today."
```

```
Option A — 귀여운 스타일 (Cute)
  "🟢안녕 🟢예쁜이들~  🟢오늘도  🔵왔어!"
    (an-nyeong ye-ppeu-ni-deul~  o-neul-do  wass-eo!)

    (an-nyeong! yeo-reo-bun! o-neul mwo hal geo-nya-myeon~)

  학생과 함께 만들기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

```
            Korean YouTubers use for their subscribers
 오늘도 = 'again today' (도 = also/again)
 왔어 = 'I came/I'm here' (casual past)

 So the full vibe is: 'Hey cuties, I'm back again today~'"

```

```
 It's like 'insane!' in English — always positive in context.

 찐이야 = 'that's the real deal' — fans use this
 when they feel something is genuinely authentic."
```

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔵 문법 도구 (Grammar Tools You Own):
  -주세요    → Please do (가주세요, 세워주세요)
  -요        → Polite ending (all strangers)
  -아요/어요  → Polite present (feelings: 감사해요, 부끄러워요)
  -(이)요    → Order ending (아아 하나 포장이요)
  -야/이야   → Casual (friends/fans: 왔어, 고마워)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚕 WEEK 1 — Taxi:
  기사님, OO(으)로 가주세요          → Please go to OO
  여기서 세워주세요                → Please stop here
  잘 모르겠어요~ / 아 네~         → Gentle deflection
  한국말 잘 몰라요~ 죄송해요~      → I don't know Korean well~

💞 WEEK 2 — Fan Interaction:
  헐! 대박!                        → OMG! Amazing!
  진짜요? 너무 감사해요 ㅠㅠ       → Really? Thank you so much
  아니에요~ 부끄러워요 ㅎㅎ        → No~ I'm shy haha
  같이 찍어요! 하나 둘 셋~        → Let's take a photo! 1 2 3!

☕ WEEK 3 — Cafe & Shopping:
  아아 하나 포장이요               → One iced americano, to go
  괜찮아요~ / 없어요~             → No thanks / I don't have one
  괜찮아요, 저 혼자 볼게요~       → I'm fine, just browsing
  이거 얼마예요?                   → How much is this?

📱 WEEK 4 — Content Creator:
  안녕 예쁜이들~ 오늘도 왔어!      → Hey cuties~ I'm back!
  봐줘서 고마워~ 사랑해요~         → Thanks for watching~ Love you~
  좋아요 구독 눌러줘~              → Hit like and subscribe~

🟡 만능 리액션 (Universal):
  아 네~     → soft yes / gentle response
  괜찮아요~  → I'm fine / No thanks (works EVERYWHERE)
  진짜요?   → Really?
  대박!     → Amazing!
  미쳤어!   → That's insane! (compliment)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
