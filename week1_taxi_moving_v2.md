## [8-25분] Target Expressions — 상황 1: 목적지 설명

### 표현 ① "OO으로 가주세요"

**SHOW IT**: 아이패드에 색상 분해

```
🟢기사님,    🟢홍대으로     🔵가주세요.
  (Driver)    (to Hongdae)    (please go)
  gi-sa-nim   hong-dae-eu-ro  ga-ju-se-yo
```

**SPLIT** (영어로 설명):
```
"기사님 = Driver (it's the respectful way to call them)
 홍대으로 = to Hongdae (으로 means 'toward')
 가주세요 = please go (가다 = go + 주세요 = please do it for me)"
```

🟡 **문화 포인트 (Why moment)**:
```
"In Korea, calling someone by their role title is the polite thing to do.
 기사님 literally means 'Mr. Driver.' It shows instant respect.
 If you just say '홍대으로 가주세요' without '기사님' — it still works,
 but adding it makes a HUGE difference. Koreans notice this."
```

**WAIT**: 학생에게 직접 말하게
```
선생: "OK, try it. Tell me to go to Gangnam."
(3초 이상 기다림...)
학생: "기사님... 강남으로... 가주세요?"
선생: "Yes! Perfect! You just told a taxi driver where to go!"
```

### 표현 ② "여기서 세워주세요"

**SHOW IT**:
```
🟢여기서      🔵세워주세요.
  (here)       (please stop)
  yeo-gi-seo   se-wo-ju-se-yo
```

**SPLIT**:
```
"여기서 = here (at this spot)
 세워주세요 = please stop (세우다 = to stop/park + 주세요)"
```

**WAIT**: 학생이 따라 말하기 (3초 여유)

### 표현 ③ 패턴 확장 (-주세요)

**SHOW IT**: 🔵 -주세요 패턴이 얼마나 강력한지 보여주기

```
🟢여기로     🔵가주세요.     → Please go here.
🟢여기서     🔵세워주세요.   → Please stop here.
🟢천천히     🔵가주세요.     → Please go slowly.
🟢오른쪽으로  🔵가주세요.     → Please go right.
```

**SPLIT**:
```
"See the pattern? You just change the FIRST part.
 The ending -주세요 stays the SAME every time.
 This is your superpower. One tool, infinite sentences."
```

**BRIDGE** →
```
"Great! Now you can tell the driver where to go.
 But here's the real challenge — what happens when the driver
 starts TALKING to you? A LOT? 😅"
```

## [25-42분] Target Expressions — 상황 2: 정중하게 대화 끊기

🟡 **문화 코칭 (Why moment — 가장 중요!)** :

**SPLIT** (영어로 깊이 설명):
```
"OK, this is a BIG cultural thing.

 In Korea, there's a strong 어르신-젊은이 (elder-younger) culture.
 Older taxi drivers see a young foreigner and think:
 'Oh, a young person alone! Let me take care of them!'

 It's not rude — it's actually CARING in Korean culture.
 They're being friendly. But for you, it feels overwhelming
 because you can't understand and you feel trapped.

 So the goal is NOT to be rude back.
 The goal is to escape GENTLY with a smile.
 Koreans are MASTERS at this. Let me teach you their moves."
```

### 방어 문장 ① "잘 모르겠어요~"

**SHOW IT**:
```
🟢잘         🔵모르겠어요~
  (well)      (I don't know)
  jal         mo-reu-gess-eo-yo

의미: "Hmm, I'm not sure~" (gentle deflection)
```

### 방어 문장 ② "한국말 잘 몰라요~ 죄송해요 😅"

**SHOW IT**:
```
🟢한국말      🟢잘     🔵몰라요~      🟡죄송해요 😅
  (Korean)    (well)   (don't know)   (sorry - cultural!)
  han-guk-mal  jal     mol-la-yo      joe-song-hae-yo
```

**SPLIT** (영어로 교정 타이밍에 맞게):
```
"죄송해요 = sorry. But here's the thing — in Korean,
 adding 죄송해요 with a smile doesn't mean you did something wrong.
 It means 'I wish I could chat with you but I can't.'
 It's the Korean way of being warm while saying no."
```

🟡 **문화 포인트**:
```
"Notice: you can flip the order!
 죄송해요. 한국말 잘 몰라요. ← also works!
 한국말 잘 몰라요. 죄송해요. ← also works!

 YOU already did this in our first lesson. Remember?
 That's because you understood the STRUCTURE, not just the words."
```

### 방어 문장 ③ "아 네~ (웃음)"

**SHOW IT**:
```
🟡아 네~  (ah ne~)
  = "Oh, yeah~" + smile

이건 문법이 아니라 🟡문화 무기다.
한국인이 대화를 부드럽게 넘길 때 쓰는 만능 반응.
```

**SPLIT**:
```
"This is the most Korean thing ever.
 아 네~ with a smile = 'I hear you but I have nothing to say'
 Koreans use this ALL THE TIME. It's not rude, it's social grace.
 Master this one sound, and you're already 50% Korean."
```

## [42-48분] Natural Upgrade + Reaction

**BRIDGE**:
```
"Nice job surviving! Now let me show you the NATURAL versions.
 What you said was correct. These upgrades make it feel more Korean."
```

**SHOW IT**: 어색한 버전 vs 자연스러운 버전 나란히

```
[교과서]  한국말 잘 모릅니다.         (too formal, feels stiff)
[자연]   한국말 잘 몰라요~ 😅        (perfect for daily life)

[교과서]  죄송합니다.                 (too serious, like you crashed the car)
[자연]   죄송해요~ 😅               (friendly sorry)

[교과서]  여기에서 정차해 주십시오.     (robot voice)
[자연]   여기서 세워주세요~           (normal human)
```

### Reaction Training

```
기사님이 뭔가 말할 때:
  🟡 "아~ 네~"        (ah~ ne~)      ← 만능 반응
  🟡 "아이고~"        (ai-go~)       ← 놀람/감탄
  🟡 "진짜요?"        (jin-jja-yo?)  ← "Really?"

이 3개만 있으면 택시에서 무적이다.
```

## [48-55분] Combination Practice ← NEW

**BRIDGE**:
```
"Awesome! Now here's the fun part.
 Let's combine what you learned and make NEW sentences."
```

**WAIT**: 학생에게 시간을 충분히 주고 기다린다.

### 조합 연습

```
선생: "OK, you know 가주세요 and 세워주세요.
      Can you tell me to go to Itaewon?" (3초+ 기다림)

학생: "기사님, 이태원으로 가주세요?"
선생: "Perfect! Now add '천천히' (slowly)." (3초+ 기다림)

학생: "기사님, 천천히... 가주세요?"
선생: "YES! You just built a sentence nobody taught you!"
```

### 방어 문장 조합

```
선생: "The driver asks you a question you don't understand.
      What do you say?" (3초+ 기다림)

학생 (자발적 조합 유도):
  "잘 모르겠어요. 한국말 잘 몰라요. 죄송해요~"
  or
  "죄송해요~ 한국말 잘 몰라요~"

선생: "Look at that — you just combined THREE phrases
      in your own order. That's not memorizing. That's SPEAKING."
```

### Don't Panic Moment 🚨

**SPLIT** (영어로 시연):
```
"Real talk: in a real taxi, you WILL freeze for a second.
 That's normal. Here's what happens in real life..."
```

**선생이 직접 시연**:
```
(기사님 역) "아이고 예쁘다~ 어디서 왔어? 모델이야?"
(학생 역 시연) "..." (1초 당황) → "아 네~ ㅎㅎ" → "죄송해요, 한국말 잘 몰라요~"

"See? The pause is FINE. Even Koreans pause.
 The 아 네~ buys you time. The 죄송해요 closes it gently.
 You are SAFE. Don't panic."
```
