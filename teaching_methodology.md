> **작성일**: 2026-04-27

> **적용 범위**: 모든 수업 (Trial, Making Friends, 확장 클래스, 오프라인 Field Trip 포함)

# 0. 한 줄 원칙

> **문법이 적(敵)이 아니다. 지루한 문법 교육이 적이다.**
> **문화 해석 + 시각적 문법 구조 = 학생이 "왜?"도 이해하고 "어떻게?"도 기억하는 수업**

---

# 1. 4대 교수법 원칙 (The Four Teaching Rules)

모든 수업에 예외 없이 적용한다. 온라인/오프라인 상관없이.

---

## Rule 1: SHOW IT (시각적 문법)

### 목표
초급 학생이 문장 속에서 자기가 뭘 배우고 있는지 **한눈에** 알게 한다.

### 색상 체계 (매 수업 동일하게 사용)

| 색상 | 용도 | 예시 |
|------|------|------|
| 🔵 파랑 | **문법 포인트** (오늘의 핵심) | 갑**니다**, 마셔**요**, 갔**어요** |
| 🟢 초록 | **어휘 / 의미 단위** | **저는** (I), **학교에** (to school) |
| 🟡 노랑 | **문화 포인트** | **밥 먹었어요?** = 안부 (not food question) |

### 실행 방법

**오프라인 수업 (아이패드)**:
1. 아이패드 필기 앱(GoodNotes, Notability 등)에서 색 펜 사용
2. 문장을 쓰고 → 학생에게 직접 따라 쓰게 함
3. 색상별로 구분하면서 구조 설명

**온라인 수업 (화면 공유)**:
1. Google Docs에서 텍스트 색상으로 구분
2. 실시간으로 작성하면서 화면 공유
3. 학생이 자기 화면에서 바로 복습 가능

**수업 후 (공통)**:
- 아이패드 필기 내용을 Google Docs로 정리
- 오프라인/온라인 모두 최종 결과물은 Google Docs에 보관
- 학생이 스마트폰으로 언제든 열어볼 수 있게

### 예시

```
오늘의 포커스: -아요/어요 (현재 정중형)

문장: 저는 커피를 마셔요.

구조 분해:
  🟢 저는    → I (topic)
  🟢 커피를  → coffee (object)
  🔵 마셔요  → drink (polite present — 오늘의 문법!)

수업 시작 멘트:
  "Today we're focusing on -아요/어요. This is the polite way
   to say things in present tense. Every verb we practice today
   will end with this."
```

### ⚠️ 경계선

이건 **교재가 아니다**. 실시간 구조화 도구다.
- 미리 만들어 놓은 교재 프린트 ❌
- 수업 중 학생과 함께 만드는 시각 자료 ✅
- 문법 설명은 1문장 이내, 나머지는 예시와 연습으로 ✅

## Rule 2: WAIT (답변 여유)

### 목표
학생의 자신감을 지키고, 생각할 시간을 주어 학습 효과를 높인다.

### 3초 규칙

```
질문한다 → 3초 이상 침묵 → 학생이 대답 → 끝까지 듣는다 → 교정
```

### ❌ 하지 말 것

```
선생: "어제 뭐 했어요?"
학생: "어제... 학교..."
선생: (바로 끼어듦) "학교에 갔어요! 갔어요가 맞아!"
```

### ✅ 이렇게 할 것

```
선생: "어제 뭐 했어요?"
(3초 이상 기다림... 학생이 생각하는 중...)
학생: "어제... 학교... 가요..."
선생: "Good job finishing the sentence! Now let's fix it together."
      "어제 = past, so we need past tense: 갔어요"
      "저는 어제 학교에 갔어요. Try it again?"
```

### 핵심
- 학생이 **틀려도 끝까지 말하게** 한다
- 말하는 중에 **메모를 조용히 적어둔다**
- 교정은 **문장이 완성된 후에**
- 교정할 때 먼저 **잘한 점을 인정**하고 → 수정

### 오프라인 Field Trip 적용
카페/식당에서 학생이 주문할 때:
- 바로 도와주지 않는다
- 학생이 시도하게 기다린다
- 막히면 귓속말로 힌트만 준다
- 끝난 후에 "잘했어! 근데 이 부분은 이렇게 하면 더 자연스러워" 교정

---

## Rule 3: BRIDGE (주제 전환)

### 목표
갑작스러운 전환으로 인한 혼란과 어색함을 방지하고, 학생이 학습 구조를 이해하게 한다.

### 전환 공식

```
[마무리] "Great job with [방금 한 것]!"
[예고]   "Now we're going to move to [다음 것]."
[연결]   (선택) "This connects to what we just learned because..."
```

### 예시

**수업 엔진 단계 전환 시:**

| 전환 | 멘트 예시 |
|------|----------|
| Warm-up → 표현 | "Good catching up! Now let's learn today's expressions." |
| 표현 → Role Play | "Now you know the phrases. Let's practice using them in a real situation." |
| Role Play → Natural Upgrade | "Nice try! Now let me show you how to say the same thing more naturally." |
| Natural Upgrade → Reaction | "The words are good. Now let's add reactions to make it feel alive." |
| Reaction → Wrap-up | "Awesome work today! Let me show you everything we covered." |

### ⚠️ 절대 하지 않을 것

```
❌ (갑자기) "자, 이제 과거형 해볼게요."
   → 학생: "...? 방금 거 끝난 건가?"

✅ "You did great with present tense! Now we're going to switch to past tense.
    This is important because when you tell stories about your day, you'll need this."
```

---

## Rule 4: SPLIT (언어 분리)

### 목표
초급 학생이 한국어 설명만으로 혼란에 빠지지 않게 하면서도, 실전 한국어 노출을 극대화한다.

### 규칙

| 상황 | 사용 언어 | 이유 |
|------|----------|------|
| 예문 말하기 | 🇰🇷 한국어 | 실전 소리에 노출 |
| 예문 해석 | 🇬🇧 영어 | 의미를 확실히 고정 |
| 문법 구조 설명 | 🇬🇧 영어 | 개념 이해 |
| 문화 맥락 설명 | 🇬🇧 영어 | "왜?" 이해 |
| Role Play 중 | 🇰🇷 한국어 | 실전 훈련 |
| Role Play 후 교정 | 🇬🇧 영어 + 🇰🇷 한국어 | 이해 + 교정 |
| 리액션 훈련 | 🇰🇷 한국어 | 입에 배게 |

### 예시

```
선생: "저는 커피를 마셔요." (🇰🇷)

선생: "This means 'I drink coffee.'" (🇬🇧)
      "저는 = I" (🇬🇧)
      "커피를 = coffee" (🇬🇧)
      "마셔요 = drink, polite present tense" (🇬🇧)

선생: "Now you try. Say 'I drink water.'" (🇬🇧)
학생: "저는... 물을... 마셔요?" (🇰🇷)
선생: "Perfect!" (🇬🇧)
```

### ❌ 하지 말 것

```
❌ "저는 커피를 마셔요 means 저는... 커피... 마셔요..."
   (한국어로만 설명 → 초급자 혼란)
```

### 단계적 전환
- 초급: 영어 70% / 한국어 30%
- 반복 수강 후: 영어 50% / 한국어 50%
- 중급 이상: 영어 30% / 한국어 70%

# 2. 실전 검증 결과 (First Lesson Evidence)

## Student #01 첫 수업 결과

### 학생 프로필
- 20세, 영국(흑인), 100만 팔로워 틱톡 모델/인플루언서
- 한국 거주, 기본 한글 읽기 가능, 회화 경험 거의 없음

### 수업 전
- 말문이 트이지 않은 상태
- 상황 전체에 대한 회화 불가능

### 수업 후 (1시간)
- **상황 전체에 대한 말문이 트임**
- 배운 표현을 **스스로 조합**해서 새로운 문장 생성

### 검증된 응용 사례

학생이 수업 중 배운 패턴을 **자발적으로 재조합**한 사례:

| 배운 표현 | 학생의 응용 | 의미 |
|----------|------------|------|
| 죄송합니다 + 한국말 잘 몰라요 | "죄송합니다. 한국말 잘 몰라요" | 순서 조합 |
| 한국말 잘 몰라요 + 죄송합니다 | "한국말 잘 몰라요. 죄송합니다" | **역순 조합** |
| -주세요 패턴 | "여기로 가주세요" / "해주세요" | **패턴 응용** |

> **이것이 의미하는 것**: 학생이 문장을 통째로 외운 게 아니라, **구조를 이해해서 조합**했다.
> 이건 Q&A 방식 + 반복(다른 스타일) + 문화 설명(Why moment)이 결합되어 나온 결과다.

### 참관 교수법 선생님 확인
- 학생의 자발적 조합 능력에 놀라움 표시
- 수업 방식의 효과를 실시간으로 확인

# 3. 계속 해야 할 것 (Validated Strengths)

교수법 선생님이 확인한 강점. 이것들은 바꾸지 않는다:

| # | 강점 | 왜 중요한가 |
|---|------|------------|
| 1 | 인내심 있는 delivery | 학생이 안전하다고 느낀다 |
| 2 | 격려 | 자신감 → 더 많이 시도 |
| 3 | 반복 + 다른 스타일 | 지루하지 않은 반복 = 장기 기억 |
| 4 | 다양한 예시 | 하나의 패턴을 여러 상황에서 이해 |
| 5 | **Why moment** | **핵심 차별화** — 다른 튜터에게 없는 것 |
| 6 | **Q&A 방식** | 질문→답변→재질문→확장 = 실전 대비 |
| 7 | 단순 문법 우선 | 필요할 때만 깊이 들어감 |
| 8 | 젠틀한 문화 강조 | 강요 없이 자연스럽게 이해 |
| 9 | 문화 차이 강조 | 프로젝트 존재 이유 |
| 10 | **Don't panic moment** | 실전 시연 + 용기 부여 |

> **특히 #5, #6, #10은 이 프로젝트만의 시그니처다. 절대 희석하지 않는다.**

# 4. 수업 엔진 업데이트 (4대 원칙 통합)

기존 6단계 수업 엔진에 4대 원칙을 통합한 버전:

## 45분 수업 (Making Friends 기준)

```
[0-5분]   Warm-up
          - 가벼운 스몰톡 (SPLIT: 영어+한국어 혼합)
          - 지난 주 복습 간단히
          - BRIDGE: "Good to see you! Let's review quickly, then learn new stuff."

[5-7분]   Today's Focus 선언
          - SHOW IT: 아이패드/Docs에 오늘의 문법 포인트 색상으로 표시
          - SPLIT: "Today we're focusing on [문법 포인트]. This is used when..."
          - BRIDGE: "OK, now let's see it in action."

[7-18분]  Target Expressions (표현 3개)
          - SHOW IT: 각 표현을 색상 구조로 분해
          - SPLIT: 한국어로 말하고 → 영어로 설명
          - WAIT: 학생에게 따라 말하게 할 때 3초 여유
          - 문화 맥락 설명 (Why moment)

[18-28분] Situation Role Play
          - BRIDGE: "Now you know the phrases. Let's use them in a real situation."
          - WAIT: 학생이 틀려도 끝까지 말하게 함
          - SPLIT: Role Play 중 = 한국어 / 교정 = 영어
          - Don't panic moment 포함

[28-35분] Natural Upgrade + Reaction
          - BRIDGE: "Nice try! Let me show you the natural version."
          - SHOW IT: 어색한 버전 vs 자연스러운 버전 나란히 기록
          - 리액션 훈련 (아이고, 진짜?, 대박)

[35-42분] 응용 연습 (Combination Practice) ← NEW
          - 오늘 배운 표현을 조합해서 새 문장 만들기
          - WAIT: 학생에게 시간 충분히 주고 기다림
          - Student #01 사례처럼 자발적 조합 유도

[42-45분] Wrap-up
          - BRIDGE: "Great work today! Let me show you what we covered."
          - SHOW IT: Google Docs 정리 화면 공유
          - 다음 수업 제안
```

# 5. 오프라인 Field Trip 교수법

오프라인 상황에도 4대 원칙을 동일하게 적용한다.

## 카페/식당 주문 시나리오

```
[준비]
  - SHOW IT: 가기 전에 아이패드에 핵심 문장 색상으로 정리
  - BRIDGE: "OK, we practiced ordering. Now let's do it for real!"

[실전]
  - WAIT: 학생이 주문 시도 → 바로 도와주지 않음
  - WAIT: 막히면 귓속말로 힌트만 ("주세요" 라고 해봐)
  - SPLIT: 힌트 = 한국어 단어만 / 설명 = 나중에 영어로

[마무리]
  - BRIDGE: "You just ordered in Korean! Let's talk about what happened."
  - 잘한 점 먼저 → 개선점
  - SHOW IT: 실전에서 나온 새 표현을 Docs에 추가
```

# 6. 색상 체계 운영 (Option C: 아이패드 + Google Docs)

## 수업 중 (아이패드)
- GoodNotes/Notability 등 필기 앱 사용
- 3가지 색 펜: 🔵 문법 / 🟢 어휘 / 🟡 문화
- 학생이 직접 따라 쓰게 함
- 수업 중 실시간으로 작성

## 수업 후 (Google Docs)
- 아이패드 필기 내용을 Google Docs로 정리
- 텍스트 색상으로 동일한 체계 유지
- 오프라인/온라인 학생 모두 Docs로 복습
- 학생 스마트폰에서 바로 접근 가능

## 이유
- 오프라인: 아이패드 직접 필기가 학습 효과 극대화
- 온라인: Google Docs 화면 공유가 유일한 시각 도구
- 복습: 두 채널 모두 최종 결과물은 Docs에 통합
- 확장: 향후 자동화 파이프라인과 연동 가능 (Google One 구독 활용)
