## [8-22분] Target Expressions — 상황 1: 카페 주문

### 🟡 문화 코칭 (Why moment)

**SPLIT** (영어):
```
"Korean ordering culture is COMPLETELY different from the UK.

 In the UK, ordering is a conversation:
 'Hi, could I possibly get a...'

 In Korea, ordering is EFFICIENT:
 '아이스 아메리카노 하나요.' DONE.

 Nobody thinks you're rude. Speed is kindness.
 The person behind you is waiting. Quick = polite.

 Also: in Korea, you pay FIRST, then get your drink.
 And most cafes now have kiosk machines (키오스크).
 But knowing the phrases helps when the machine breaks
 or when you go to a small local cafe."
```

### 표현 ① "아이스 아메리카노 하나 포장이요."

**SHOW IT**:
```
🟢아이스 아메리카노   🟢하나    🟢포장      🔵이요.
  (iced americano)     (one)    (takeout)    (polite marker)
  a-i-seu a-me-ri-ka-no  ha-na   po-jang     i-yo

→ "One iced americano, to go."
```

**SPLIT**:
```
"This is the simplest order formula in Korean:
 [음료] + [개수] + [포장/먹고 갈게요] + 이요.

 포장이요 = takeout (to go)
 먹고 갈게요 = for here (I'll eat/drink here)
 (meok-go gal-ge-yo)

 That's it. No 'could I possibly' needed."
```

**WAIT**: 학생 연습 (3초+)
```
선생: "Order me an iced latte, to go."
(3초+ 기다림)
학생: "아이스 라떼 하나 포장이요."
선생: "Perfect Korean ordering. You're done in 3 seconds."
```

### 표현 ② 알바생의 3대 질문 해독

**SHOW IT**: 알바생이 무조건 묻는 3가지

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔵 알바생 3대 질문 (You WILL hear these)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: "🔵드시고 가시나요? 🔵포장이요?"
     (deu-si-go ga-si-na-yo? po-jang-i-yo?)
     → "For here or to go?"
     답: "포장이요." or "먹고 갈게요."

Q2: "🔵영수증 드릴까요?"
     (yeong-su-jeung deu-ril-kka-yo?)
     → "Do you need a receipt?"
     답: "괜찮아요~" (gwaen-cha-na-yo) = "No thanks~"

Q3: "🔵포인트 적립하시나요?"
     (po-in-teu jeok-rip-ha-si-na-yo?)
     → "Do you have a loyalty card?"
     답: "없어요~" (eop-seo-yo) = "I don't have one~"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**SPLIT**:
```
"These 3 questions come EVERY SINGLE TIME.
 Once you know them, you'll never freeze at a cafe again.

 The best part? Your answers are ONE WORD each:
 포장이요 / 괜찮아요 / 없어요

 THREE words. That's your entire cafe survival kit."
```

**WAIT**: 각 질문에 대해 학생 연습

```
선생: "드시고 가시나요?"  (WAIT 3초+)
학생: "포장이요."
선생: "영수증 드릴까요?"  (WAIT 3초+)
학생: "괜찮아요~"
선생: "포인트 적립하시나요?" (WAIT 3초+)
학생: "없어요~"
선생: "Three for three. You survived the cafe! ☕"
```

## [22-32분] Target Expressions — 상황 2: 올리브영/쇼핑

**BRIDGE**:
```
"Great job with cafes! Now let's talk about shopping.
 This one has a different challenge: shop assistants."
```

### 🟡 문화 코칭 (Why moment)

**SPLIT** (영어):
```
"In Korean shops — especially places like 올리브영 (Olive Young,
 it's like a Korean Sephora/Boots) — staff will come up to you
 and START EXPLAINING products.

 This isn't annoying. It's their JOB.
 In Korean service culture, leaving a customer alone = BAD service.
 They're trained to approach you.

 So you need a GENTLE way to say 'I'm fine, just browsing.'
 Without being rude. The Korean way."
```

### 표현 ③ "괜찮아요, 저 혼자 볼게요~ (미소)"

**SHOW IT**:
```
🟢괜찮아요,       🟢저    🟢혼자    🔵볼게요~    🟡(미소)
  (It's OK)       (I)    (alone)   (I'll look)   (smile!)
  gwaen-cha-na-yo  jeo   hon-ja    bol-ge-yo

→ "I'm fine, I'll just look around by myself~ (smile)"
```

**SPLIT**:
```
"괜찮아요 = 'It's OK' / 'No thank you' / 'I'm fine'
 This word is your SWISS ARMY KNIFE in Korean.
 It works in 100 different situations.

 저 혼자 볼게요 = 'I'll browse alone'
 볼게요 = 'I'll look' (future tense, soft)

 The 미소 (smile) at the end is KEY.
 Without the smile, it sounds cold.
 With the smile, it sounds warm and Korean."
```

**WAIT**: 학생 연습
```
선생 (점원 역): "안녕하세요~ 뭐 찾으시는 거 있으세요?"
(WAIT 3초+)
학생: "괜찮아요, 저 혼자 볼게요~" (미소)
선생: "Perfect! The smile makes all the difference."
```

### 표현 ④ "이거 얼마예요?" (가격 묻기)

**SHOW IT**:
```
🟢이거      🔵얼마예요?
  (this)     (how much?)
  i-geo      eol-ma-ye-yo?
```

**SPLIT**:
```
"이거 = this (pointing at something)
 얼마예요? = how much is it?

 Simple. Point + ask. Works everywhere."
```

## [42-47분] Natural Upgrade + Reaction

**BRIDGE**:
```
"Nice work! Now let me show you how to sound
 even more natural — like a local."
```

**SHOW IT**: 자연스러운 업그레이드

```
[초급 - 지금]                    [자연 - 목표]
아이스 아메리카노 하나 포장이요   →  아아 하나 포장이요
                                    (아아 = 아이스 아메리카노의 줄임말!)
                                    (a-a = Korean slang for iced americano)

괜찮아요                        →  아 괜찮아요~ 감사해요~
                                    (add 감사해요 = warmer)

없어요                          →  아 없어요~ 괜찮아요~
                                    (softer with 아 and ~)
```

🟡 **문화 포인트**:
```
"아아 (a-a) is how ALL Koreans order iced americano.
 If you say '아아 하나요' at a cafe, the barista will do a double take.
 'Wait... this foreigner knows 아아?!'
 It's an instant connection moment."
```

```
"Real talk: at a cafe, sometimes they say something

 Here's your emergency kit:
 1. Smile
 2. '괜찮아요~' (covers 90% of unexpected questions)
 3. If they keep asking: '잘 모르겠어요~ 죄송해요~'

 This is your SAFETY NET. You're never stuck."
```
