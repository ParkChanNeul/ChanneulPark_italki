# Q/A Drill Sets

이 폴더는 질문을 듣고 바로 답하는 훈련용 Quizlet companion set을 관리합니다.

Reference 세트는 단어와 표현을 빠르게 확인하는 용도이고, Q/A Drill 세트는 실제 수업처럼 질문과 답변을 한 카드에서 반복하는 용도입니다.

## 사용법

1. `../import/qna/`의 `.tsv` 파일을 연다.
2. 파일 전체를 복사한다.
3. Quizlet "만들기" > "가져오기"에 붙여넣기.
4. 설정은 용어-정의 구분 `Tab`, 카드 구분 `줄바꿈`으로 둔다.
